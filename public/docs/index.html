Introduction

Base URL: http://localhost:1001

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

Authenticate requests to this API's endpoints by sending an Authorization header with the value "Bearer {YOUR_API_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

object

string

Relationships to include (comma separated). Allowed values: genres, companies, keywords, cast, crew, user.

Example:
genres,cast
fields
string

Fields to include (comma separated). Allowed values: id, title, tagline, description, poster, budget, revenue, runtime, popularity, vote_average, vote_count, imdb_id, homepage, release_date.

sort
string

Fields to sort by (comma separated) (prefix - for descending). Allowed values: id, title, runtime, popularity, release_date.

curl --request GET \

    --header "Accept: application/json"
const url = new URL(
    "http://localhost:1001/api/movies"
);

const params = {
    "page": "1",
    "per_page": "15",
    "include": "genres,cast",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {YOUR_API_KEY}",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->get(
    'http://localhost:1001/api/movies',
    [
        'headers' => [
            'Authorization' => 'Bearer {YOUR_API_KEY}',
            'Accept' => 'application/json',
        ],
        'query' => [
            'page' => '1',
            'per_page' => '15',
            'include' => 'genres,cast',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
